## Descripción del proyecto
Este es el juego tradicional de la serpiente. La idea principal del mismo es mover mediante los botones de flecha del teclado, la serpiente a lo largo y ancho del tablero, con la finalidad de comer la mayor cantidad de comida posible.

Se debe de evitar chocar con las paredes y con el propio cuerpo de la serpiente.


## Detalle técnicos
Básicamente la idea de este proyecto fue la de aprender y practicar los conocimientos asociados a Canvas y JavaScript ES6.
